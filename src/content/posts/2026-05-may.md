This is post zero — the devlog exists. Going forward, one post per month covering what shipped, what broke, and what's next.

## Where things stand

Critter is a co-op multiplayer climbing exploration game on Unreal Engine 5 + Lyra. The current state, at a glance:

- **Climbing system** — ledge detection, stamina, vault transitions wired through GAS
- **Missions and shards** — Rift Shard collection drives biome unlocks
- **Plugins** — Cog (debug tools), Voxel, Redpoint EOS for online services
- **Tooling** — heavy Claude Code investment: a tree of skills for blueprint editing, materials, audio, GAS scaffolding, and a fleet of subagents for research and post-implementation checks

## What this devlog is

Each monthly post is generated from the actual git history of the **Critter** game repo, then hand-edited with screenshots and commentary. The drafting is a Claude Code skill (`/devlog-draft`), so the source of truth is always commits — no aspirational roadmaps, no "soon™" entries that quietly disappear.
